# 微助教签到助手

一个面向 Windows 的本地签到辅助工具，已经补上适合普通用户的一键启动方案。

## 快速开始

1. 双击仓库根目录里的 `一键启动.bat`
2. 等待脚本自动创建虚拟环境、安装依赖并启动服务
3. 浏览器会自动打开 `http://127.0.0.1:5000/`

默认会自动尝试获取 OpenID 并持续监听签到任务。

## 不会命令行也能用

- 首页会实时展示当前状态、签到任务和二维码结果
- 自动获取 OpenID 失败时，可以直接在页面里粘贴 OpenID 或完整链接
- 需要结束服务时，双击 `停止服务.bat`

## 环境要求

- Windows
- Python 3
- 微信桌面端可正常打开

## 仓库里的启动相关文件

- `一键启动.bat`：推荐给普通用户，双击即可
- `停止服务.bat`：停止后台服务
- `start_web_app.ps1`：启动脚本本体
- `stop_web_app.ps1`：停止脚本本体

## 进阶说明

仓库仍然保留了 `mitmproxy` 和 CV 抓取相关脚本，适合调试或高级场景使用。普通用户优先使用默认首页流程即可。
